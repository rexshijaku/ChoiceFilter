ChoiceFilter


Dropdown - Radio | Dependency

An example where a select dropdown filters the choices of a group of radio buttons.


Example 1

In this example dependent choices have their own wrappers. Note that div elements with class 'wrapperClass' wrapp the choices which will be filtered. Each filtered choice will be unchecked during filter.

In this example dependent choices have not their own wrappers. In this case ChoiceFilter.js is only capable to find out their corresponding labels, this is done by using the id attribute of the choices and for attribute of labels. Each filtered choice will be unchecked during filter.
